Spring Core
=========================

DI
--------------

Java Connfigによる実装例は以下の通り。メソッド名がBean名、
戻り値が、DI時にそのBeanのインスタンスとして登録される。

    .. sourcecode:: bash    
        :linenos:

        @Configuration
        public class AppConfig {
            @Bean
            public UserRepository userRepository(){
                return new UserRepositoryImpl();
            }
        }
